# ADR-002

## Décision

Utiliser Google Sheets comme interface de collecte.

## Statut

Acceptée

## Contexte

Les opérateurs rencontrent des difficultés dans la gestion des fichiers locaux.

## Justification

Google Sheets permet :

- une source unique ;
- la collaboration ;
- l'historisation ;
- la réduction des erreurs de version.

## Conséquences

La collecte de données reposera sur Google Drive.
