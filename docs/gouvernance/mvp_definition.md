# MVP Definition

## Version

V1

## Objectif

Remplacer les calculs Excel manuels par une plateforme d'analyse centralisée.

## Inclus

### Collecte

- Google Sheets
- Validation des données

### Stockage

- PostgreSQL

### Modélisation

- Star Schema

### Reporting

- Dashboard Stock
- Dashboard Consommation

### Qualité

- Contrôle doublons
- Contrôle index
- Contrôle kilométrage

## Exclus

### Sécurité avancée

- Authentification

### Prévisions

- Forecasting

### Alertes

- Notifications

### Application Web

- Interface dédiée

### Traçabilité avancée

- Audit (historique des modifications)
- Historisation des transactions

## Critère de réussite

Production automatique des KPI de stock et consommation.
