# User Guide

Documentation destinée aux utilisateurs finaux et aux personnes installant FuelSight.

## Objectif

Permettre à un utilisateur ou un nouvel arrivant d'installer, configurer et utiliser FuelSight sans assistance.

## Structure

- [Installation Guide](installation.md) - prérequis, clonage, environnement virtuel, Docker
- [Configuration Guide](configuration.md) - variables d'environnement, Google Sheets, Power BI
- [Quick Start](quick_start.md) - mise en route en moins de 10 minutes
- [User Manual](user_manual.md) - actions possibles par rôle (Fuel Operator, Fleet Manager, Data Engineer)
- [FAQ](faq.md) - questions fréquentes

Ce dossier est complémentaire de [`CONTRIBUTING.md`](../../CONTRIBUTING.md) (racine du projet), qui s'adresse aux contributeurs plutôt qu'aux utilisateurs finaux.
