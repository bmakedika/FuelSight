# FAQ

## Où saisir les transactions ?

Dans Google Sheets.

---

## Les données sont-elles modifiées ?

Non.

Les données RAW sont conservées sans modification.

---

## Comment corriger une erreur ?

Une nouvelle transaction de correction est enregistrée.

---

## Où consulter les indicateurs ?

Dans Power BI.

---

## Les données historiques peuvent-elles être supprimées ?

Non.

FuelSight applique une stratégie d'historisation des données.
