# User Manual

## Fuel Operator

Le Fuel Operator peut :

- enregistrer une réception
- enregistrer une distribution
- enregistrer un transfert

---

## Fleet Manager

Le Fleet Manager peut :

- consulter les KPI
- suivre les stocks
- analyser les consommations
- détecter les écarts

---

## Data Engineer

Le Data Engineer peut :

- exécuter les pipelines
- contrôler la qualité
- consulter les journaux
