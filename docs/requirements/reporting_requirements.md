# Reporting Requirements

## Objectif

Décrire les exigences applicables aux tableaux de bord et aux indicateurs.

---

# RR-001 - Dashboard exécutif

Le dashboard doit afficher :

- stock théorique ;
- stock physique ;
- écart de stock ;
- consommation mensuelle.

---

# RR-002 - Dashboard consommation

Le dashboard doit permettre l'analyse :

- par véhicule ;
- par camion ;
- par générateur ;
- par site.

---

# RR-003 - Dashboard contrôle

Le dashboard doit afficher :

- anomalies ;
- écarts d'index ;
- écarts de stock.

---

# RR-004 - Filtres

Tous les dashboards doivent proposer :

- Date
- Site
- Asset
- Type d'asset

---

# RR-005 - Rafraîchissement

Les tableaux de bord doivent être mis à jour après chaque exécution réussie du pipeline ELT.

---

# RR-006 - Export

Les utilisateurs doivent pouvoir exporter les visualisations et les données selon les fonctionnalités offertes par Power BI.
