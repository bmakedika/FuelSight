# Product

Documents orientés produit : utilisateurs, scénarios d'usage et indicateurs.

## Objectif

Décrire qui utilise FuelSight, comment, et quels indicateurs le produit doit fournir.

## Structure

- [Personas](personas.md) - Fuel Operator, Fleet Manager, Data Engineer
- [Use Cases](use_cases.md) - scénarios d'utilisation détaillés (UC-001, UC-002...)
- [Functional Specifications](functional_specifications.md) - vue narrative des fonctionnalités, organisée par domaine et rattachée aux `FR-XXX` de `requirements/functional_requirements.md`
- [KPI Catalog](kpi_catalog.md) - fiche détaillée par indicateur (calcul, source, dashboard, fréquence)
