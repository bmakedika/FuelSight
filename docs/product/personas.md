# Personas

## Objectif

Décrire les principaux utilisateurs de FuelSight, leurs responsabilités, leurs besoins et leurs attentes afin de guider les choix fonctionnels du produit.

---

# Persona 1 - Fuel Operator

## Description

Le Fuel Operator est chargé des opérations quotidiennes liées au carburant.

Il saisit les mouvements de carburant et réalise les contrôles de stock.

## Responsabilités

- Enregistrer les réceptions
- Enregistrer les distributions
- Enregistrer les transferts
- Réaliser les contrôles tank
- Vérifier les données saisies

## Objectifs

- Gagner du temps lors de la saisie
- Réduire les erreurs
- Disposer d'une interface simple

## Difficultés actuelles

- Plusieurs fichiers Excel
- Calculs manuels
- Risque d'erreurs

---

# Persona 2 - Fleet Manager

## Description

Responsable de la gestion opérationnelle de la flotte et des stocks carburant.

## Responsabilités

- Contrôler les stocks
- Valider les données
- Suivre les consommations
- Identifier les anomalies

## Objectifs

- Disposer de KPI fiables
- Identifier rapidement les écarts
- Réduire les ruptures de stock

---

# Persona 3 - Data Engineer

## Description

Responsable de la plateforme de données FuelSight.

## Responsabilités

- Construire les pipelines
- Garantir la qualité des données
- Superviser les traitements
- Maintenir le Data Warehouse

## Objectifs

- Pipeline fiable
- Traçabilité
- Automatisation
