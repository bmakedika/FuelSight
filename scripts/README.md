# Scripts

Ce dossier contient les scripts utilisés par FuelSight.

## Objectif

Automatiser les opérations liées au traitement des données.

## Structure

### Ingestion

- Export Google Sheets vers CSV
- Chargement CSV vers PostgreSQL

### Transformation

- Nettoyage des données
- Enrichissement
- Contrôles qualité

### Maintenance

- Sauvegarde
- Archivage
- Surveillance

## Convention de nommage

- `extract_*.py`
- `load_*.py`
- `transform_*.py`
- `validate_*.py`

## Technologies

- Python
- Bash
- SQL
